        - homepage stylesheet
    - _chic.scss_
        - menu "Le chic à la française" stylesheet
    - _delice.scss_
        - menu "Le délice des papilles" stylesheet
    - _note.scss_
        - menu "La note enchantée" stylesheet
    - _palette.scss_
        - menu "Lapalette du goût" stylesheet

        - mixins
    - _responsive.scss_
        - breakpoints & responsive mixin
